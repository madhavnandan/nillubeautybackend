## Setup
1. Install dependencies: `npm install`
2. Create MySQL database and run the SQL in `init_db.sql` or use the provided `create_tables.sql`
3. Copy `.env.example` to `.env` and fill in values.
4. Start: `npm start` (or `npm run dev` with nodemon)

## Notes
- This is a starter backend with endpoints for auth, products, services, transactions, and simple P&L.
- Designed for development use. Do not run in production without securing secrets and hardening.
